
## 📥 Debian Netinst ISO ডাউনলোড লিংক

ডেবিয়ানের অফিসিয়াল ওয়েবসাইট থেকে সরাসরি নিচের লিংকে ক্লিক করে ৩২০-৪০০ মেগাবাইটের ISO ফাইলটি ডাউনলোড করে নিন:

* **ডাউনলোড লিংক:** [Debian Netinst 64-bit (PC/Intel/AMD) Official ISO](https://get.debian.org/images/archive/)

---

## 🛠️ VMware-এ Debian Netinst সেটআপ করার সম্পূর্ণ নিয়ম

ভার্চুয়াল মেশিনে ইন্সটল করার সময় আমাদের মূল লক্ষ্য থাকবে **কোনো ডেস্কটপ এনভায়রনমেন্ট (GUI) ইন্সটল না করা**, যাতে সাইজ একদম কম থাকে।

### ধাপ ১: নতুন ভার্চুয়াল মেশিন তৈরি (VMware Settings)

1. VMware ওপেন করে **Create a New Virtual Machine**-এ ক্লিক করুন।
2. **Installer disc image file (iso)** অপশনটি সিলেক্ট করে ডাউনলোড করা ডেবিয়ান ISO ফাইলটি দেখিয়ে দিন।
3. Guest OS হিসেবে **Linux** এবং Version হিসেবে **Debian 12 (64-bit)** সিলেক্ট করুন।
4. র‍্যাম (RAM) দিন **৫১২ এমবি অথবা ১ জিবি** (১ জিবির বেশি দেওয়ার কোনো প্রয়োজন নেই)।
5. হার্ডডিস্ক সাইজ **১০ জিবি বা ২০ জিবি** দিন (ইন্সটলের পর এটি মাত্র ১-২ জিবি জায়গা নেবে)।
6. সবশেষে **Finish** দিয়ে ভার্চুয়াল মেশিনটি অন (Power On) করুন।

---

### ধাপ ২: মূল ইন্সটলেশন প্রসেস (ধাপে ধাপে)

1. **বুট মেনু:** ভার্চুয়াল মেশিন অন হলে একটি বুট মেনু আসবে। সেখান থেকে **Graphical Install** সিলেক্ট করে `Enter` চাপুন।
2. **ভাষা ও অঞ্চল:** Language হিসেবে `English`, Location হিসেবে `Bangladesh` (অথবা আপনার পছন্দমতো), এবং Keyboard হিসেবে `American English` সিলেক্ট করে এগিয়ে যান।
3. **নেটওয়ার্ক সেটআপ:** সিস্টেম স্বয়ংক্রিয়ভাবে নেটওয়ার্ক কনফিগার করবে। এরপর `Hostname` চাবে, সেখানে `debian` বা আপনার ফার্মের নাম দিতে পারেন। `Domain name` ফাঁকা রেখে `Continue` দিন।
4. **পাসওয়ার্ড সেটআপ (খুবই গুরুত্বপূর্ণ):** * প্রথমে **Root password** চাবে। এটি হচ্ছে মেইন অ্যাডমিন পাসওয়ার্ড। একটি কঠিন পাসওয়ার্ড দিন এবং মনে রাখুন।
* এরপর **Full name for the new user** চাবে, এখানে একটি সাধারণ নাম দিন (যেমন: `farmuser`)।
* এই সাধারণ ইউজারের জন্য একটি আলাদা পাসওয়ার্ড দিন। (ইউজার এই পাসওয়ার্ড দিয়ে লগইন করবে, কিন্তু কোনো কিছু আনইন্সটল করতে পারবে না)।


5. **ডিস্ক পার্টিশন:** এখানে **Guided - use entire disk** সিলেক্ট করুন। পরের স্ক্রিনে আপনার ভার্চুয়াল ডিস্কটি সিলেক্ট করে `All files in one partition` দিন। সবশেষে **Finish partitioning and write changes to disk** সিলেক্ট করে `Yes` দিন।

---

### ধাপ ৩: সফটওয়্যার সিলেকশন (সবচেয়ে গুরুত্বপূর্ণ অংশ)

ইন্সটলেশনের শেষের দিকে সিস্টেম কিছু বেসিক ফাইল ডাউনলোড করবে এবং স্ক্রিনে **Software Selection** নামের একটি পেজ আসবে। **এই জায়গাটি খুব সাবধানে খেয়াল করুন:**

1. স্ক্রিনে `Debian desktop environment`, `GNOME`, `... XFCE`, `... LXDE` এরকম অনেকগুলো অপশন দেখতে পাবেন যেগুলোর পাশে টিক চিহ্ন দেওয়া থাকবে।
2. স্পেসবার (Spacebar) চেপে **ডেস্কটপ এনভায়রনমেন্টের সবকটি টিক চিহ্ন তুলে দিন।** (কোনো প্রকার গ্রাফিকাল ডেস্কটপ আমরা নেব না)।
3. শুধুমাত্র নিচের ২টি অপশনে টিক চিহ্ন রাখবেন:
* **`standard system utilities`**
* **`SSH server`** (ভবিষ্যতে রিমোটলি কাজ করার সুবিধার জন্য)


4. টিক দেওয়া শেষ হলে `Continue` চাপুন।

---

### ধাপ ৪: ইন্সটলেশন সম্পন্ন করা

1. সবশেষে আপনার কাছে জানতে চাইবে **Install the GRUB boot loader to your primary drive?**, এখানে **`Yes`** সিলেক্ট করুন।
2. পরের স্ক্রিনে `/dev/sda` সিলেক্ট করে `Enter` দিন।
3. কিছুক্ষণের মধ্যে স্ক্রিনে দেখাবে `Installation is complete`। **Continue**-এ ক্লিক করলে ভার্চুয়াল মেশিনটি রিস্টার্ট হবে।

---

## 🎯 ইন্সটলেশন শেষে আপনি কী দেখতে পাবেন?

কম্পিউটার রিস্টার্ট হওয়ার পর আপনি উইন্ডোজের মতো কোনো মাউস বা সুন্দর স্ক্রিন দেখতে পাবেন না। স্ক্রিনে একটি কালো টার্মিনাল আসবে এবং লেখা থাকবে:
`debian login:`

এখানে আপনি আপনার তৈরি করা সাধারণ ইউজারনেম (`farmuser`) এবং পাসওয়ার্ড দিয়ে লগইন করতে পারবেন। আপনার মিনিমাল ডেবিয়ান লিনাক্স এখন সম্পূর্ণ প্রস্তুত!


এই ধাপে আমরা কালো স্ক্রিনের ভেতর **Apache, PHP, MySQL (MariaDB)** এবং আমাদের **মিনিমাল গ্রাফিক্স ইঞ্জিনসহ ক্রোমিয়াম ব্রাউজার** ইন্সটল করব।

ধাপগুলো নিচে একদম সহজভাবে দেওয়া হলো:

---

## ধাপ ১: সিস্টেমে লগইন এবং আপডেট করা

১. প্রথমে কালো স্ক্রিনে `debian login:` এর জায়গায় **`root`** লিখুন এবং আপনার দেওয়া রুট (অ্যাডমিন) পাসওয়ার্ডটি দিয়ে লগইন করুন। (মনে রাখবেন, লিনাক্সে পাসওয়ার্ড টাইপ করার সময় স্ক্রিনে কোনো অক্ষর বা স্টার `*` দেখায় না, তাই সাবধানে টাইপ করে `Enter` চাপবেন)।
২. লগইন করার পর সিস্টেমে ইন্টারনেট কানেকশন সচল আছে কিনা তা নিশ্চিত করতে এবং সিস্টেম আপডেট করতে এই কমান্ডটি লিখে `Enter` দিন:

```bash
apt update && apt upgrade -y

```

---

## ধাপ ২: লোকালহোস্ট সার্ভার (ZAMP এর বিকল্প) ইন্সটল করা

এবার আমরা আপনার পিএইচপি প্রোজেক্টটি ইন্টারনেট ছাড়া রান করার জন্য সার্ভার ইন্সটল করব। টার্মিনালে নিচের কমান্ডটি লিখুন:

```bash
apt install apache2 php libapache2-mod-php mariadb-server php-mysql -y

```

> **এই কমান্ডটি কী করবে?**
> * এটি ব্যাকগ্রাউন্ডে Apache ওয়েব সার্ভার এবং PHP চালু করে দেবে।
> * MariaDB (যা MySQL-এর লেটেস্ট রূপ) ডাটাবেজ ইন্সটল করে দেবে।
> * এটি উইন্ডোজের জ্যাম্পের মতোই কাজ করবে, কিন্তু ব্যাকগ্রাউন্ডে সাইজ নেবে মাত্র কয়েক মেগাবাইট।
> 
>

## ধাপ ৩: গ্রাফিক্স ইঞ্জিন ইন্সটল করা

যেহেতু আমাদের সিস্টেমে কোনো ভারী ডেস্কটপ বা ফাইল ম্যানেজার নেই, তাই সরাসরি কালো স্ক্রিন থেকে একটি ব্রাউজার স্ক্রিনে ফুটিয়ে তোলার জন্য আমাদের একটি ছোট্ট গ্রাফিক্স ইঞ্জিন (Xorg) লাগবে। টার্মিনালে এই কমান্ডটি লিখুন:

```bash
apt install xinit xserver-xorg

```

*(এটি ইন্সটল হতে ১-২ মিনিট সময় নিতে পারে, একটু অপেক্ষা করবেন।)*

---

## ধাপ ৪: আপনার পিএইচপি ফাইল এবং ডাটাবেজ কোথায় রাখবেন?

সার্ভার তো ইন্সটল হলো, এবার আপনার ফার্মের ক্যালকুলেটর অ্যাপের কোডগুলো রাখার পালা:

1. **PHP ফাইল:** আপনার পিএইচপি প্রজেক্টের সব ফাইল লিনাক্সের এই ফোল্ডারে পেস্ট করে দেবেন: `/var/www/html/`
2. **Database:** ডাটাবেজ ইমপোর্ট বা তৈরি করার জন্য টার্মিনালে `mysql` লিখলেই ডাটাবেজ কনসোল ওপেন হয়ে যাবে। সেখানে আপনি উইন্ডোজের মতোই `CREATE DATABASE farm_db;` লিখে আপনার ডাটাবেজ তৈরি করতে পারবেন।

## 🛠️ ফাইল ম্যানেজার ইন্সটল করার কমান্ড

আপনার ডেবিয়ানের কালো স্ক্রিনে `root` ইউজার দিয়ে লগইন থাকা অবস্থায় নিচের কমান্ডটি লিখে `Enter` দিন:

```bash
apt install pcmanfm lxterminal -y

```

> **এখানে কী কী ইন্সটল হলো?**
> * `pcmanfm`: এটি হচ্ছে আপনার গ্রাফিকাল ফাইল ম্যানেজার (GUI File Manager)।
> * `lxterminal`: এটি একটি হালকা গ্রাফিকাল টার্মিনাল। ফাইল ম্যানেজারের ভেতরে থেকে যদি কখনো কোনো কমান্ড দেওয়া লাগে, তখন এটি কাজে লাগবে।
> 
>

## 📂 ফাইল ম্যানেজারটি কীভাবে ওপেন করবেন?

যেহেতু আমাদের সিস্টেমে কোনো "স্টার্ট মেনু" বা ডেস্কটপ আইকন নেই, তাই ফাইল ম্যানেজারটি গ্রাফিকালি ওপেন করার জন্য টার্মিনালে এই কমান্ডটি লিখুন:

```bash
startx /usr/bin/pcmanfm

```

**ম্যাজিক দেখুন!** কমান্ডটি দেওয়ার সাথে সাথে কালো স্ক্রিন চলে যাবে এবং আপনার সামনে উইন্ডোজের মতো সুন্দর একটি ফাইল ম্যানেজার ওপেন হয়ে যাবে। সেখানে আপনি মাউস ব্যবহার করে ক্লিক করে করে যেকোনো ফোল্ডারে ঢুকতে পারবেন।

## ⚠️ একটি গুরুত্বপূর্ণ সমস্যা ও তার সমাধান (Permission Issue)

আপনি যখন ফাইল ম্যানেজার দিয়ে আপনার পিএইচপি ফাইলগুলো `/var/www/html/` ফোল্ডারে রাখতে যাবেন, তখন লিনাক্স আপনাকে **"Permission Denied"** বা ফাইল কপি করতে বাধা দিতে পারে। কারণ লিনাক্সে এই ফোল্ডারটি অত্যন্ত সুরক্ষিত থাকে।

এটি চিরতরে সমাধান করার জন্য (যাতে আপনি সহজেই ফাইল কাট, কপি, পেস্ট করতে পারেন), ফাইল ম্যানেজারটি বন্ধ করে মেইন কালো টার্মিনালে এসে এই কমান্ডটি দিয়ে রাখুন:

```bash
sudo chmod -R 777 /var/www/html/

```

এই কমান্ডটি দেওয়ার পর আপনি যেকোনো সময় ফাইল ম্যানেজার ওপেন করে আপনার পিএইচপি প্রোজেক্টের সব ফাইল (যেমন: `index.php`, `style.css` ইত্যাদি) খুব সহজে মাউস দিয়ে ড্র্যাগ অ্যান্ড ড্রপ বা কপি-পেস্ট করে `/var/www/html/` ফোল্ডারে রেখে দিতে পারবেন।

## 🛠️ ধাপ ১: Firefox ইন্সটল করার কমান্ড

টার্মিনালে এই কমান্ডটি লিখে `Enter` দিন:

```bash
apt install firefox-esr -y

```

> **নোট:** লিনাক্সে `firefox-esr` হলো ফায়ারফক্সের সবচেয়ে স্টেবল (Stable) এবং লাইটওয়েট সংস্করণ, যা কম হার্ডওয়্যারে চমৎকার কাজ করে।

## 🚀 ধাপ ২: Firefox-কে সব ফিচারসহ রান করার কমান্ড

এবার ওপেনবক্স উইন্ডো ম্যানেজারের সাহায্যে ফায়ারফক্সকে স্বাভাবিক মোডে (ক্লোজ, মিনিমাইজ বাটন ও অ্যাড্রেস বারসহ) চালু করার জন্য নিচের কমান্ডটি দিন:

```bash
startx /usr/bin/openbox-session -- /usr/bin/firefox-esr http://localhost

```

## 🔒 ধাপ ১: পাসওয়ার্ড ছাড়া অটো-লগইন সেটআপ (Auto-Login)

ইউজার যাতে কম্পিউটার অন করার পর `debian login:` বা পাসওয়ার্ডের স্ক্রিন না দেখে সরাসরি সিস্টেমে ঢুকে যায়, তার জন্য এই কনফিগারেশনটি করতে হবে:

১. টার্মিনালে এই কমান্ডটি লিখে `Enter` দিন:

```bash
systemctl edit getty@tty1

```

২. একটি খালি এডিটর ওপেন হবে। সেখানে নিচের ৩টি লাইন হুবহু পেস্ট করে দিন:

```text
[Service]
ExecStart=
ExecStart=-/sbin/agetty --autologin root --noclear %I $TERM

```

*(কিবোর্ডের `Ctrl + O` চেপে `Enter` দিন সেভ করার জন্য, তারপর `Ctrl + X` চেপে বের হয়ে আসুন।)*

## 🚀 ধাপ ২: অন হওয়ামাত্র গ্রাফিক্স ইঞ্জিন চালু করা

সিস্টেম অটো-লগইন হওয়ার সাথে সাথে যাতে কালো স্ক্রিন থেকে সরাসরি গ্রাফিক্স মোডে চলে যায়, তার ব্যবস্থা করব:

১. টার্মিনালে লিখুন:

```bash
nano ~/.bash_profile

```

২. ফাইলের একদম নিচের লাইনে এই কোডটুকু লিখে দিন:

```bash
if [ -z "$DISPLAY" ] && [ "$XDG_VTNR" -eq 1 ]; then
  exec startx
fi

```

*(কিবোর্ডের `Ctrl + O` চেপে `Enter` দিন সেভ করার জন্য, তারপর `Ctrl + X` চেপে বের হয়ে আসুন।)*

## 🌍 ধাপ ৩: ফায়ারফক্সকে অটো-স্টার্টআপে দেওয়া (ফাইনাল ম্যাজিক)

এবার আমরা ওপেনবক্সকে বলে দেব যে, গ্রাফিক্স অন হওয়ামাত্র তুমি ফায়ারফক্সকে তার হোম স্ক্রিনসহ চালু করে দাও।

১. টার্মিনালে প্রথমে এই কমান্ডটি দিন যাতে ওপেনবক্সের কনফিগারেশন ফোল্ডার তৈরি হয়:

```bash
mkdir -p ~/.config/openbox

```

২. এবার অটোস্টার্ট ফাইলটি এডিট করার জন্য লিখুন:

```bash
nano ~/.config/openbox/autostart

```

৩. এই খালি ফাইলে নিচের লাইনটি লিখে দিন:

```bash
firefox-esr http://localhost &

```

> **💡 ট্রিক:** আপনি যদি চান ফায়ারফক্স কোনো নির্দিষ্ট ওয়েবসাইট না খুলে তার ডিফল্ট হোম স্ক্রিন (যেখানে গুগল সার্চ বা শর্টকাট থাকে) নিয়ে ওপেন হোক, তাহলে শুধু `firefox-esr &` লিখবেন।

*(একইভাবে `Ctrl + O` চেপে `Enter` এবং `Ctrl + X` চেপে বের হয়ে আসুন।)*

### ধাপ ১: ফাইল বা ফোল্ডার জিপ (Zip) বা আনজিপ (Unzip) প্যাকেজ ইন্সটল করা

টার্মিনালে নিচের কমান্ডটি লিখে `Enter` দিন:

```bash
apt install zip unzip xarchiver -y

```

### 📂 এখন এটি কীভাবে ব্যবহার করবেন?

সবকিছু সেটআপ হয়ে গেছে! এখন ম্যাজিক দেখার পালা:

* **আনজিপ করার নিয়ম:** ফাইল ম্যানেজারে গিয়ে যেকোনো `.zip` ফাইলের ওপর মাউসের **Right-Click (ডান ক্লিক)** করুন। সেখানে **"Extract Here"** বা **"Extract To..."** অপশন দেখতে পাবেন। ওটাতে ক্লিক করলেই ফাইলটি আনজিপ হয়ে যাবে।
* **জিপ করার নিয়ম:** যেকোনো ফোল্ডার বা ফাইলের ওপর **Right-Click** করে **"Create Archive..."** বা **"Compress..."** অপশনে ক্লিক করুন। এরপর নাম দিয়ে `.zip` সিলেক্ট করলেই সেটি জিপ ফাইলে রূপান্তরিত হয়ে যাবে।

### 💡 একটি প্রো-টিপ (ভবিষ্যতের জন্য):

আপনার পিএইচপি প্রোজেক্টের জিপ ফাইল যদি উইন্ডোজ থেকে পেনড্রাইভে করে এনে এই লিনাক্সের `/var/www/html/` ফোল্ডারে আনজিপ করেন, তবে আনজিপ করার পর ফাইলগুলোর ওপর আবার সেই পারমিশন কমান্ডটি দিয়ে দেবেন, যাতে ব্রাউজার বা ফাইল ম্যানেজার কোনো বাধার সৃষ্টি না করে:

```bash
sudo chmod -R 777 /var/www/html/
```

লিনাক্স দুনিয়ায় যেকোনো চলমান বা ইনস্টল করা অপারেটিং সিস্টেমকে (Debian, Ubuntu, Arch, Fedora ইত্যাদি) হুবহু বুটেবল ISO ফাইলে রূপান্তর করার জন্য সবচেয়ে আধুনিক এবং জনপ্রিয় ইউনিভার্সাল (Universal) টুল হলো **Penguins-Eggs** (সংক্ষেপে এটিকে **eggs** বলা হয়)। 

এই টুলটি আপনার সম্পূর্ণ লিনাক্স সিস্টেমের একটি স্ন্যাপশট নিয়ে একটি লাইভ ও বুটেবল ISO তৈরি করে। এই ISO-র ভেতরে একটি গ্রাফিক্যাল ইনস্টলার (**Calamares**) যুক্ত থাকে। ফলে আপনি পরবর্তীতে যেকোনো কম্পিউটারে সাধারণ অপারেটিং সিস্টেমের মতোই মাত্র কয়েকটি ক্লিকের মাধ্যমে আপনার এই কাস্টমাইজড ওএসটি ইনস্টল করতে পারবেন।

নিচে ধাপ অনুসারে (Step-by-Step) এই কাজটি করার নিয়মটি দেওয়া হলো:

---

### ধাপ ১: সিস্টেম পরিষ্কার করা (System Cleanup)
আপনার তৈরি করা ISO ফাইলের সাইজ যাতে অনেক বড় না হয়ে যায়, সেজন্য অপ্রয়োজনীয় ক্যাশ ফাইল এবং টেম্পোরারি ফাইলগুলো ডিলিট করে নেওয়া ভালো।
* আপনার টার্মিনালটি ওপেন করুন।
* ওএস যদি **Ubuntu/Debian** ভিত্তিক হয়, তবে এই কমান্ডগুলো রান করুন:
  ```bash
  sudo apt clean
  sudo apt autoremove
  ```
* ওএস যদি **Arch Linux** ভিত্তিক হয়, তবে রান করুন:
  ```bash
  sudo pacman -Scc
  ```

### ধাপ ২: 'git' ইনস্টল করা
টুলটি ডাউনলোড করার জন্য সিস্টেমে `git` থাকা প্রয়োজন। না থাকলে নিচের কমান্ড দিয়ে ইনস্টল করে নিন:
* **Ubuntu/Debian-এর জন্য:**
  ```bash
  sudo apt update && sudo apt install git -y
  ```
* **Arch Linux-এর জন্য:**
  ```bash
  sudo pacman -S git
  ```

### ধাপ ৩: Penguins-Eggs ইনস্টল করা
এবার ইউনিভার্সাল ইনস্টলেশন স্ক্রিপ্টের সাহায্যে টুলটি ইনস্টল করতে নিচের কমান্ডগুলো একে একে টার্মিনালে রান করুন:
```bash
git clone https://github.com/pieroproietti/get-eggs
cd get-eggs
sudo ./get-eggs
```
*(এই স্ক্রিপ্টটি আপনার লিনাক্স ডিস্ট্রিবিউশনটি চিহ্নিত করে স্বয়ংক্রিয়ভাবে প্রয়োজনীয় সব ডিপেন্ডেন্সি ও কনফিগারেশন সেটআপ করে দেবে।)*

### ধাপ ৪: টুলটি প্রথমবার কনফিগার করা
ইনস্টলেশন সফলভাবে শেষ হওয়ার পর, সিস্টেমের সাথে টুলটির মেলবন্ধন তৈরি করতে নিচের কমান্ডটি রান করুন:
```bash
sudo eggs dad
```
*(এটি স্ক্রিনে কিছু অপশন দেখাবে, সাধারণত এন্টার চেপে ডিফল্ট সেটিংসগুলো সিলেক্ট করে গেলেই চলে।)*

### ধাপ ৫: কাস্টম বুটেবল ISO তৈরি করা
এখন আপনার কাস্টমাইজড লিনাক্স ও ইনস্টল করা অ্যাপগুলো সহ মূল ISO ফাইলটি তৈরি করার জন্য নিচের কমান্ডটি দিন:
```bash
sudo eggs produce
```
* **সময়:** এই প্রক্রিয়াটি আপনার ভার্চুয়াল মেশিনের প্রসেসর, র‍্যাম এবং আপনার ইনস্টল করা অ্যাপগুলোর সাইজের ওপর ভিত্তি করে ১০ থেকে ৩০ মিনিট (বা তার বেশি) সময় নিতে পারে। প্রক্রিয়াটি চলাকালীন টার্মিনাল বন্ধ করবেন না।
* কাজ শেষ হলে স্ক্রিনে আপনাকে জানানো হবে যে ISO তৈরি সম্পন্ন হয়েছে।

### ধাপ ৬: তৈরি হওয়া ISO ফাইলটি সংরক্ষণ করা
তৈরি হওয়া বুটেবল ISO ফাইলটি সাধারণত আপনার কম্পিউটারের `/home/eggs/` ডিরেক্টরিতে সংরক্ষিত থাকে।
* ফাইল ম্যানেজার দিয়ে `/home/eggs/` ফোল্ডারে যান (অথবা টার্মিনালে `ls /home/eggs` লিখে চেক করুন)।
* সেখানে আপনার ওএস-এর নাম সংবলিত একটি `.iso` ফাইল দেখতে পাবেন।
* এই ISO ফাইলটি ভার্চুয়াল মেশিন (VMware) থেকে কপি করে আপনার উইন্ডোজের মূল হার্ডডিস্কে নিয়ে রাখুন।

### ধাপ ৭: অন্য যেকোনো ডিভাইসে ওএস-টি ইনস্টল করার নিয়ম
১. **পেনড্রাইভ বুটেবল করুন:** Rufus, BalenaEtcher বা Ventoy সফটওয়্যার ব্যবহার করে আপনার তৈরি করা ISO ফাইলটি একটি পেনড্রাইভে রাইট (Flash) করে নিন।
২. **বুট করুন:** পেনড্রাইভটি যেকোনো ফিজিক্যাল পিসিতে (বা অন্য কোনো ভার্চুয়াল মেশিনে) প্লাগ-ইন করে কম্পিউটারটি বুট মেনু থেকে পেনড্রাইভের মাধ্যমে বুট করুন।
৩. **লাইভ রান ও ইনস্টল:** সিস্টেমটি প্রথমে লাইভ মোডে চালু হবে। ডেস্কটপে আপনি **"Install System" (Calamares)** নামক একটি আইকন দেখতে পাবেন। সেখানে ডাবল ক্লিক করে খুব সাধারণ উইজার্ডের মাধ্যমে আপনার আগে থেকে তৈরি করা কাস্টম ওএস-টি নতুন কম্পিউটারে ইনস্টল করে নিতে পারবেন।

*(নোট: লাইভ সিস্টেমে বুট করার পর ইউজারনেম এবং পাসওয়ার্ড চাইলে সাধারণত ডিফল্ট হিসেবে ইউজারনেম `live` এবং পাসওয়ার্ড `evolution` থাকে।)*
